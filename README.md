# FilePresenter

File presenter is a Phoenix Applicaiton that will help you to show any file changes by passing a path directory, all changes are going to be shown in real time. This tools can be used for helping people with workshops related with coding.

<img src="file_presenter_logo.png"/>

## Setup

  * Install dependencies with `mix deps.get`
  * Install Node.js dependencies with `npm install`


## Usage

    PORT=4001 WATCH=~/my/project mix phx.server

## Note

It will ignore all files listed in the .gitignore directory
